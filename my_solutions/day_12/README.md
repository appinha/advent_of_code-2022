# Day 12: Hill Climbing Algorithm

## Summary

📃 **Input:** a heightmap of the surrounding area.

⭐ **Part One:** find the fewest steps required to move from the current positions to the highest location.

⭐ **Part Two:** find the fewest steps required to move starting from any square with the lowest elevation to the highest location.
