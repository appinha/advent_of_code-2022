# Day 22: Monkey Map

## Summary

📃 **Input:** a map of the board (a 2D grid representation of a cube) and the path you must follow (an alphanumeric string).

⭐ **Part One:** follow the path given in the monkeys' notes to find the final password.

⭐ **Part Two:** fold the map into a cube, then follow the path given in the monkeys' notes to find the final password.
